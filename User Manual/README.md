- Multilevel models,
- Survival models with frailty,
- Dynamic panel-data (DPD) regressions,
- SEM (structural equation modeling),
- Binary, count, and censored outcomes,
- ARCH,
- Multiple imputation,
- Latent class analysis (LCA),
- Treatment effects,
- Survey data,
- Bayesian analysis,
- Finite mixture models (FMM)
- More
